# ABE-SHR
Shared Health Record Service for Records encrypted using ABE

## Setup
1. Edit the following files:
    * ```nginx/abe-shr-nginx```
    * ```nginx/abe-shr-nginx.conf```
    * ```upstart/abe-shr-upstart.conf```
    * ```config.py```
    * ```install.sh```
2. Run ```install.sh```
    ```
    ./install.sh
    ```
